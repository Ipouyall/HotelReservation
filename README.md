# HotelReservation

Socket Programming,First CA of The Computer Networks course @ University of Tehran, spring 2023

## How to run project

### 01. Install Requirements

On macos, run:

```shell
brew install cmake
```

For linux:

```shell
sudo apt-get install cmake
```

### 02. Generate Makefile

Run:

```shell
cmake .
```

### Build project

Run:

```shell
make
```
